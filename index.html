Set Theory & Metric Spaces

Weeks 1–2 · Exercises 1.2, 1.3, Problem 1(a)–(d)

This is your warm-up territory. Even if you know real analysis, Søren's proofs here establish the notation and style used throughout the course.

1.1 Essential Set Operations

The following identities are used constantly. Know their proofs by heart.

IdentityWhat it says
\(A \setminus B = A \cap B^c\)Set difference as intersection with complement
\(\left(\bigcup A_i\right)^c = \bigcap A_i^c\)De Morgan: complement of union
\(\left(\bigcap A_i\right)^c = \bigcup A_i^c\)De Morgan: complement of intersection
\(f^{-1}(H^c) = f^{-1}(H)^c\)Preimages commute with complements
\(f^{-1}\!\left(\bigcup A_i\right) = \bigcup f^{-1}(A_i)\)Preimages commute with arbitrary unions
\(f^{-1}\!\left(\bigcap A_i\right) = \bigcap f^{-1}(A_i)\)Preimages commute with arbitrary intersections
Key Insight
Preimages behave perfectly under all set operations. This is why measurability is defined via preimages — it makes the theory work cleanly.

1.2 The Sup-Norm vs Euclidean Norm (Exercise 1.2)

Exercise 1.2 establishes the equivalence of \(\rho_\infty(x,y) = \max_i |x_i - y_i|\) and \(\rho_2(x,y)\). The key inequalities are:

\[ \rho_\infty(x,y) \;\le\; \rho_2(x,y) \;\le\; \sqrt{d}\,\rho_\infty(x,y) \]

Left inequality. Let \(j\) achieve the max. Then \(\rho_\infty = |x_j - y_j| \le \sqrt{\sum |x_i - y_i|^2} = \rho_2\).

Right inequality. Each \((x_i-y_i)^2 \le \rho_\infty^2\), so summing \(d\) terms gives \(\rho_2 \le \sqrt{d}\,\rho_\infty\).

Consequence (1.2(e)). These two metrics generate the same topology on \(\mathbb{R}^d\).

1.3 Density of ℚ and ℝ∖ℚ (Exercise 1.3)

  • ℚ is dense: For \(x < y\), find \(n\) with \(n(y-x)>1\), then \(m\) with \(nx < m \le nx+1\). Then \(x < m/n < y\).
  • ℝ∖ℚ is dense: \(\mathbb{Q}+\pi\) is dense in \(\mathbb{R}\), and rational + irrational = irrational.
  • Extension to \(\mathbb{R}^d\): Use \(\rho_\infty\) — approximate each coordinate independently.

1.4 Sequential Characterisation of Limits (Exercise 1.19)

Theorem
\(\lim_{t\to a} F(t) = c\) if and only if \(F(t_n)\to c\) for every sequence \(t_n \to a\) with \(t_n \ne a\).
Exam Technique
This is used in almost every continuity proof. When proving continuity of an integral (e.g. Exercise 5.23), reduce to sequences and apply Dominated Convergence.

σ-Algebras & Borel Sets

Weeks 2–4 · Exercises 1.5, 1.6, 1.7, 1.9, 1.10, 1.11, 1.19, 1.20, 1.23, 1.24

2.1 Definition & Key Examples

A σ-algebra \(\mathcal{E}\) on a set \(X\) satisfies: (σ1) \(X \in \mathcal{E}\); (σ2) \(A\in\mathcal{E}\Rightarrow A^c\in\mathcal{E}\); (σ3) countable unions of elements of \(\mathcal{E}\) are in \(\mathcal{E}\).

σ-algebraContains
\(\mathcal{P}(X)\) — power setEvery subset. All functions are measurable.
\(\{\emptyset, X\}\) — trivialOnly \(\emptyset\) and \(X\). Only constant functions are measurable.
\(\mathcal{B}(\mathbb{R})\) — BorelGenerated by open sets; contains all "reasonable" subsets of \(\mathbb{R}\).
\(\{A\subseteq\mathbb{R} : A\text{ or }A^c\text{ countable}\}\)The countable/co-countable σ-algebra. NOT countably generated (Ex 1.10).

2.2 The Borel σ-Algebra (Exercises 1.5, 1.6)

\(\mathcal{B}(\mathbb{R})\) is generated by any of the following (know how to prove equality):

  • Open sets \(G \subseteq \mathbb{R}\)
  • Closed sets \(F \subseteq \mathbb{R}\)
  • Half-open intervals \((a,b]\) for \(a,b\in\mathbb{R}\)
  • Half-open intervals \((a,b]\) for \(a,b\in\mathbb{Q}\) (use density of ℚ)
Standard Proof Technique
To show \(\sigma(\mathcal{A}) = \sigma(\mathcal{B})\): show \(\mathcal{A}\subseteq\sigma(\mathcal{B})\) and \(\mathcal{B}\subseteq\sigma(\mathcal{A})\). By monotonicity of \(\sigma(\cdot)\), equality follows.

2.3 Algebra vs σ-Algebra (Exercise 1.7)

\(\mathcal{A} = \{A\subseteq\mathbb{R} : |A|<\infty \text{ or }|A^c|<\infty\}\) is a set algebra but not a σ-algebra. The set \(\mathbb{N} = \bigcup_{n}\{n\}\) is a countable union of finite (hence measurable) sets, yet both \(\mathbb{N}\) and \(\mathbb{N}^c\) are infinite, so \(\mathbb{N}\notin\mathcal{A}\).

Key Distinction
An algebra only requires finite union stability. A σ-algebra requires countable union stability. This is what makes Lebesgue integration possible — you can take limits of measurable sets.

2.4 Generated σ-Algebras (Exercise 1.9)

Given \(A_1=\{1,2\}\), \(A_2=\{3,4\}\), \(A_3=\{2,3,4\}\) on \(X=\{1,2,3,4,5,\ldots\}\): identify atomic pieces \(B_1=\{1\}\), \(B_2=\{2\}\), \(B_3=\{3,4\}\). Each \(A_i\) is a union of \(B_j\)'s, and enumerating from the atoms is easier.

2.5 Limsup and Liminf of Sets (Exercise 1.11)

\[\liminf_n A_n \;=\; \bigcup_{N=1}^{\infty}\bigcap_{n=N}^{\infty} A_n \;=\; \{x : x\in A_n \text{ for all sufficiently large }n\}\] \[\limsup_n A_n \;=\; \bigcap_{N=1}^{\infty}\bigcup_{n=N}^{\infty} A_n \;=\; \{x : x\in A_n \text{ for infinitely many }n\}\]

Always: \(\liminf A_n \subseteq \limsup A_n\). For increasing \((A_n)\): both equal \(\bigcup A_n\). For decreasing: both equal \(\bigcap A_n\).

2.6 The Countable/Co-countable σ-Algebra is Not Countably Generated (Ex 1.10)

Let \(\mathcal{E} = \{A\subseteq\mathbb{R}: A\text{ or }A^c\text{ countable}\}\). Suppose for contradiction \(\mathcal{E}=\sigma(\{A'_n\})\) with each \(A'_n\) countable. For any \(x\) outside the union of all \(A'_n\), the singleton \(\{x\}\in\mathcal{E}\) cannot be in \(\sigma(\{A'_n\})\) — a contradiction.

The Moral
\(\mathcal{B}(\mathbb{R})\) is countably generated (by \((a,b]\) with \(a,b\in\mathbb{Q}\)), but \(\mathcal{E}\) is not — yet \(\mathcal{E}\subseteq\mathcal{B}(\mathbb{R})\). Countable generation does NOT pass to sub-σ-algebras.
03
Measures

Weeks 2–4 · Exercises 1.12, 1.15, 1.17, 1.18, 1.20, 1.23, 1.24

3.1 Definition and Basic Properties

A measure \(\mu\) on \((X,\mathcal{E})\) satisfies (m1) \(\mu(\emptyset)=0\) and (m2) countable additivity: \(\mu\!\left(\bigsqcup_n A_n\right)=\sum_n\mu(A_n)\) for pairwise disjoint \(A_n\).

PropertyStatement
Finite additivity\(\mu(A\cup B)=\mu(A)+\mu(B)\) for \(A,B\) disjoint
Monotonicity\(A\subseteq B \Rightarrow \mu(A)\le\mu(B)\)
Excision\(A\subseteq B,\;\mu(A)<\infty \Rightarrow \mu(B\setminus A)=\mu(B)-\mu(A)\)
Subadditivity\(\mu\!\left(\bigcup A_n\right)\le\sum\mu(A_n)\)
Continuity from below\(A_n\uparrow A \Rightarrow \mu(A_n)\to\mu(A)\)
Continuity from above\(A_n\downarrow A,\;\mu(A_1)<\infty \Rightarrow \mu(A_n)\to\mu(A)\)
Critical
Continuity from above requires \(\mu(A_1)<\infty\). Without this the theorem fails. Counterexample: \(A_n=(n,\infty)\) with Lebesgue measure — then \(\bigcap A_n=\emptyset\) but \(\mu(A_n)=\infty\) for all \(n\).

3.2 Inclusion-Exclusion (Exercise 1.12)

\[ \mu(A\cup B) = \mu(A) + \mu(B) - \mu(A\cap B) \]

Write \(A\cup B = A\sqcup(B\setminus(A\cap B))\), apply finite additivity, then excision (requires \(\mu(A\cap B)<\infty\)).

3.3 Linear Combinations of Measures (Exercise 1.15)

If \(\mu_1,\mu_2,\ldots\) are measures and \(a_n\ge 0\), then \(\mu=\sum_n a_n\mu_n\) is also a measure. Proof for countable additivity: interchange the two sums using nonnegativity (Tonelli for series).

3.4 Alternative Characterisations (Exercise 1.23)

A set function with \(\mu(\emptyset)=0\) is a measure iff it is finitely additive and continuous from below. For finite measures, continuity from below can be replaced by continuity from above.

3.5 The Distribution Function (Exercise 1.24)

For a finite measure \(\mu\) on \((\mathbb{R},\mathcal{B}(\mathbb{R}))\), define \(F_\mu(x)=\mu\!\left((-\infty,x]\right)\). This is increasing, right-continuous (use continuity from above on \((-\infty,x+\tfrac{1}{n}]\downarrow(-\infty,x]\)), and continuous at \(x\) iff \(\mu(\{x\})=0\).

3.6 The Borel-Cantelli Lemma (Exercise 1.18)

Borel-Cantelli Lemma
If \(\displaystyle\sum_{n=1}^{\infty}\mu(A_n)<\infty\), then \(\mu\!\left(\limsup_n A_n\right)=0\).

Proof. The tail \(\sum_{k=n}^{\infty}\mu(A_k)\to 0\). By subadditivity, \(\mu\!\left(\bigcup_{k\ge n}A_k\right)\le\sum_{k\ge n}\mu(A_k)\to 0\). Since \(\limsup A_n=\bigcap_n\bigcup_{k\ge n}A_k\) is an intersection of decreasing sets, \(\mu(\limsup A_n)\le\mu\!\left(\bigcup_{k\ge n}A_k\right)\to 0\) for all \(n\).

Measurable Functions

Weeks 4–5 · Exercises 4.4, 4.5, 4.8, 4.9, 4.10, 4.13, 4.16, 4.17

4.1 Definition & Equivalent Conditions

\(f:(X,\mathcal{E})\to(\mathbb{R},\mathcal{B}(\mathbb{R}))\) is measurable iff \(f^{-1}(B)\in\mathcal{E}\) for all \(B\in\mathcal{B}(\mathbb{R})\). It suffices to check generators. Equivalent conditions:

  1. \(\{f > t\}\in\mathcal{E}\) for all \(t\in\mathbb{R}\)
  2. \(\{f \ge t\}\in\mathcal{E}\) for all \(t\in\mathbb{R}\)
  3. \(\{f < t\}\in\mathcal{E}\) for all \(t\in\mathbb{R}\)
  4. \(\{f \le t\}\in\mathcal{E}\) for all \(t\in\mathbb{R}\)

4.2 Compositions and Operations

OperationResult
\(f\) meas., \(g\) meas.\(g\circ f\) measurable
\(f, g\) real meas.\(f+g,\; f-g,\; fg,\; |f|\) all measurable
\(f_n\) meas. for all \(n\)\(\sup f_n,\;\inf f_n,\;\limsup f_n,\;\liminf f_n\) measurable
\(f_n\to f\) pointwise\(f\) measurable (Cor 4.3.7)

4.3 The Standard Machine (Exercise 4.16)

Standard Machine
To prove a property \(P\) holds for all \(f\in\mathcal{M}(\mathcal{E})^+\), it suffices to show:
  1. \(P\) holds for \(\mathbf{1}_A\) for all \(A\in\mathcal{E}\)
  2. \(P\) is preserved under nonneg. linear combinations
  3. \(P\) is preserved under increasing pointwise limits
Then \(P\) holds for all \(f\in\mathcal{M}(\mathcal{E})^+\) by the simple-function approximation theorem.
Key Use
Exercise 4.17 applies this to show \(f\circ\phi\) is measurable: set \(\mathcal{V}=\{f\circ\phi : f\in\mathcal{M}(\mathcal{F})^+\}\) and verify (i)–(iii) using \(\mathbf{1}_B\circ\phi = \mathbf{1}_{\phi^{-1}(B)}\).

The Lebesgue Integral

Weeks 5–8 · Exercises 5.2, 5.3, 5.5, 5.9, 5.11, 5.12, 5.14, 5.16, 5.17, 5.19, 5.23, 5.25, 5.26

5.1 The Three Convergence Theorems

Monotone Convergence (MCT)
If \(0\le f_n\uparrow f\) pointwise, then \(\displaystyle\int f_n\,d\mu\to\int f\,d\mu\). (The limit may be \(+\infty\).)
Fatou's Lemma
If \(f_n\ge 0\), then \(\displaystyle\int\liminf_n f_n\,d\mu \;\le\; \liminf_n\int f_n\,d\mu\).
Dominated Convergence (DCT)
If \(f_n\to f\) pointwise, \(|f_n|\le g\) for all \(n\), and \(\int g\,d\mu<\infty\), then \(\displaystyle\int f_n\,d\mu\to\int f\,d\mu\).
TheoremWhen to use
MCTFunctions nonneg. and increasing. Limit can be infinite.
DCTFunctions not necessarily monotone. Need integrable dominant. Limit must be finite a.e.
FatouFunctions nonneg. You only get an inequality, not equality.

5.2 Integration of Simple Functions (Exercise 5.2)

For \(s = \sum_{j=1}^n a_j\mathbf{1}_{A_j}\) in standard form (disjoint \(A_j\)):

\[ \int s\,d\mu = \sum_{j=1}^n a_j\,\mu(A_j) \]

Example (5.2(c)): \(s(x)=\sum_{n=1}^\infty \frac{1}{n^2}\mathbf{1}_{(n-1,n]}\). Then \(\int s\,d\lambda = \sum_{n=1}^\infty \frac{1}{n^2}=\frac{\pi^2}{6}\).

5.3 The Dirac Measure (Exercise 5.3)

\[ \int f\,d\delta_a = f(a) \quad\text{for all }f\in\mathcal{M}(\mathcal{P}(X))^+ \]

Proof via Standard Machine: \(\int\mathbf{1}_A\,d\delta_a=\delta_a(A)=\mathbf{1}_A(a)\). Extend by linearity and MCT.

5.4 When Interchange Fails (Exercise 5.14)

Let \(f_n = n\cdot\mathbf{1}_{(0,1/n]}\). Then \(f_n\to 0\) pointwise, but \(\int f_n\,d\lambda = 1\) for all \(n\). So \(\int\lim f_n = 0\ne 1 = \lim\int f_n\).

  • DCT fails: Any dominant \(g\ge f_n\) satisfies \(\int g\ge\sum 1/n = \infty\).
  • MCT fails: The sequence is not increasing — \(f_1(2/3)=1>0=f_2(2/3)\).
Lesson
Always verify the hypotheses. Without an integrable dominant (DCT) or monotonicity (MCT), interchange can fail.

5.5 Null Sets and A.E. Properties (Exercise 5.9)

  • Every countable set is a Lebesgue null set.
  • If \(N\) is null, then \(\mathbb{R}\setminus N\) is dense in \(\mathbb{R}\).
  • If \(f,g\) continuous and \(f=g\) \(\lambda\)-a.e., then \(f=g\) everywhere (by density).

5.6 L¹ and the Counting Measure (Exercise 5.12)

For the counting measure \(\tau\) on \(\mathbb{N}\): \(\int f\,d\tau = \sum_{n=1}^\infty f(n)\). Therefore \(L^1(\tau) = \{\text{absolutely convergent sequences}\}\). The Lebesgue integral does NOT see conditional convergence — it requires absolute convergence.

5.7 Differentiation Under the Integral (Exercises 5.23, 5.25)

Continuity (Ex 5.23)
Let \(F(t)=\int_X f(x,t)\,\mu(dx)\). If each \(f(x,\cdot)\) is continuous at \(t_0\) and \(|f(x,t)|\le g(x)\) for all \(t\) with \(g\) integrable, then \(F\) is continuous at \(t_0\).
Differentiability (Ex 5.25)
Under differentiability of \(f(x,\cdot)\) and integrability of \(\partial_t f\): \[F'(t) = \int_X \frac{\partial}{\partial t}f(x,t)\,\mu(dx)\] Proof: bound difference quotients by MVT, then apply DCT.

Product Measures & Independence

Weeks 7–8 · Exercises 2.1, 2.2, 6.1, 6.4, 6.5, 6.7, 6.8

6.1 Independence (Exercises 2.1, 2.2)

Events \(A\) and \(B\) are independent if \(\mu(A\cap B)=\mu(A)\cdot\mu(B)\). The system of sets independent of a class \(\mathcal{J}\) (denoted \(\mathcal{U}_\mathcal{J}\)) forms a Dynkin system but is generally not a σ-algebra (Ex 2.1(b): two sets can each be independent of \(B\) without their intersection being so).

6.2 Fubini and Tonelli

Tonelli (nonneg. functions, σ-finite measures)
For \(f\in\mathcal{M}(\mathcal{E}\otimes\mathcal{F})^+\): \[\int\!\!\int f\,d(\mu\otimes\nu) = \int\!\left[\int f(x,y)\,\nu(dy)\right]\mu(dx) = \int\!\left[\int f(x,y)\,\mu(dx)\right]\nu(dy)\]
Fubini (integrable functions)
Same conclusion for \(f\in L^1(\mu\otimes\nu)\).
Warning
Exercise 6.7 shows Tonelli fails without σ-finiteness: with \(\tau=\) counting measure on \(\mathbb{R}\), swapping integration order for \(\mathbf{1}_\Delta\) (the diagonal) gives \(0\ne\infty\).

6.3 Cavalieri's Principle (Exercises 6.1, 6.5)

\[ \lambda_2(S) = \int_{\mathbb{R}} \lambda_1(S_x)\,\lambda_1(dx), \quad S_x=\{y:(x,y)\in S\} \]

Example (6.1): Unit disk \(D\) has slice \(D_x=[-\sqrt{1-x^2},\sqrt{1-x^2}]\) of length \(2\sqrt{1-x^2}\). So \(\lambda_2(D)=\int_{-1}^1 2\sqrt{1-x^2}\,dx=\pi\).

Weeks 9–10 · Exercises 7.2, 7.3, 7.5, 7.6, 7.10, 7.11, 7.16, 7.18

7.1 Jensen's Inequality

Jensen
If \(\mu(X)=1\), \(f\in L^1(\mu)\), and \(\varphi\) is convex on an interval containing the range of \(f\), then: \[\varphi\!\left(\int f\,d\mu\right) \;\le\; \int \varphi\circ f\,d\mu\]

Exercise 7.5 derives the weighted power mean inequality: for \(\ell_i>0\) and convex \(\varphi\),

\[ \varphi\!\left(\frac{\sum\ell_i x_i}{\sum\ell_i}\right) \;\le\; \frac{\sum\ell_i\,\varphi(x_i)}{\sum\ell_i} \]

Proof: define \(\mu=\sum\frac{\ell_j}{\sum\ell_i}\delta_{x_j}\), a probability measure. Apply Jensen.

7.2 Markov's Inequality (Exercise 7.3)

Markov
For \(\psi\) increasing with \(\psi(t)>0\): \[\mu(\{|f|\ge t\}) \;\le\; \frac{1}{\psi(t)}\int\psi(|f|)\,d\mu\]

Proof: on \(\{|f|\ge t\}\), we have \(1\le\psi(|f|)/\psi(t)\). Integrate both sides. Special case \(\psi(t)=t^p\) gives Chebyshev-type inequalities.

Hilbert Spaces

Weeks 9 & 11–12 · Exercises 9.2, 9.3, 9.6, 9.8, 9.13, 9.19

8.1 Inner Products and Cauchy-Schwarz

Cauchy-Schwarz
\(|\langle u,v\rangle| \le \|u\|\cdot\|v\|\). Equality iff \(u\) and \(v\) are linearly dependent.

Exercise 9.2: \(\|u\| = \sup\{|\langle u,v\rangle| : \|v\|\le 1\}\). The sup is achieved at \(v=u/\|u\|\).

8.2 Orthogonal Complements (Exercise 9.6)

For a subspace \(V\subseteq H\), \(V^\perp=\{u\in H:\langle u,v\rangle=0\;\forall v\in V\}\).

Double Complement
\((V^\perp)^\perp = \overline{V}\) (the closure of \(V\)). For closed subspaces: \((U^\perp)^\perp = U\).

8.3 Bounded Linear Operators (Exercise 9.19)

\(T:V\to V\) is bounded iff any of: \(T\) is continuous; \(\sup_{\|x\|\le 1}\|Tx\|<\infty\); \(\|Tx\|\le K\|x\|\) for some \(K\). The operator norm \(\|T\|_{\mathrm{op}}=\sup_{\|x\|\le 1}\|Tx\|\) satisfies \(\|S\circ T\|_{\mathrm{op}}\le\|S\|_{\mathrm{op}}\|T\|_{\mathrm{op}}\).

8.4 Orthonormal Bases and Parseval (Exercise 9.8)

Parseval's Formula
If \((e_n)\) is an orthonormal basis of \(H\): \[\|x\|^2 = \sum_n |\langle x,e_n\rangle|^2, \qquad \langle x,y\rangle = \sum_n\langle x,e_n\rangle\overline{\langle y,e_n\rangle}\]

Complex Integration & Fourier Analysis

Weeks 10–14 · Exercises 8.1, 8.2, 8.3, 11.4, 11.5, 12.2, 12.4, 12.5, 12.7

9.1 Complex-Valued Integration (Exercises 8.1, 8.2)

For \(f=g+ih\), integrate real and imaginary parts separately. Key computation (Ex 8.2): for \(z\in\mathbb{C}\) with \(\mathrm{Re}(z)<0\),

\[ \int_0^\infty e^{xz}\,d\lambda(x) = -\frac{1}{z} \]

9.2 The Gaussian Integral (Exercise 11.4)

The Gaussian Integral
\[\int_{\mathbb{R}} e^{-x^2}\,d\lambda = \sqrt{\pi}, \qquad \int_{\mathbb{R}} e^{-x^2/2}\,d\lambda = \sqrt{2\pi}\]

Proof by polar coordinates.

  1. Write \(I^2 = \int_{\mathbb{R}^2} e^{-(x^2+y^2)}\,d\lambda_2\) (Tonelli).
  2. Use the co-area formula; in polar coordinates with density \(f(r)=2\pi r\) on \((0,\infty)\):
  3. \(I^2 = \int_0^\infty 2\pi r\, e^{-r^2}\,dr = \pi\) (substitution \(u=r^2\)).
  4. Hence \(I=\sqrt{\pi}\).

Exercise 11.5 proves the Fourier transform of the Gaussian by showing \(F(t)=\int\cos(tx)e^{-x^2/2}\,d\lambda\) satisfies \(F'(t)=-tF(t)\) with \(F(0)=\sqrt{2\pi}\), giving \(F(t)=\sqrt{2\pi}\,e^{-t^2/2}\).

9.3 The Fourier Transform (Exercises 12.1–12.5)

Definition
For \(f\in L^1(\lambda)\): \[\hat{f}(t) = \frac{1}{\sqrt{2\pi}}\int_{\mathbb{R}} e^{-itx}f(x)\,d\lambda(x)\]

Key properties:

  • \(\|\hat{f}\|_\infty \le \frac{1}{\sqrt{2\pi}}\|f\|_1\) and \(\hat{f}\) is continuous.
  • Differentiation: If \(f'\in L^1\), then \(\widehat{f'}(t) = it\,\hat{f}(t)\).
  • Convolution: \(\widehat{f*g} = \sqrt{2\pi}\,\hat{f}\cdot\hat{g}\).

Exercise 12.4 — Fourier transform of the Poisson kernel \(H_\sigma(x)=e^{-\sigma|x|}\):

\[ \hat{H}_\sigma(t) = \sqrt{\frac{2}{\pi}}\cdot\frac{\sigma}{\sigma^2+t^2} \]

9.4 The Inversion Theorem and Convolution Equations (Exercises 12.5, 12.7)

Inversion Theorem (12.4.2)
If \(f,\hat{f}\in L^1\), then for \(\lambda\)-a.e. \(x\): \[f(x) = \frac{1}{\sqrt{2\pi}}\int_{\mathbb{R}} e^{itx}\hat{f}(t)\,d\lambda(t)\]

Exercise 12.5: \(C^2_c\) functions satisfy \(\hat{f}\in L^1\) via \(|t^2\hat{f}(t)|\le\|f''\|_1\), so \(1/t^2\) integrability on \(|t|>1\) gives \(\hat{f}\in L^1\), and the inversion theorem applies pointwise.

Exercise 12.7: Solve \(f-f''=u\) via convolution with the Green's function \(G=\tfrac{1}{2}e^{-|x|}\). Solution: \(f=\tfrac{1}{2}(H_1*u)\).

Radon-Nikodym & Change of Variables

Weeks 11–12 · Exercises 10.5, 11.4

Radon-Nikodym Theorem
If \(\nu\ll\mu\) (ν is absolutely continuous w.r.t. μ), then there exists measurable \(f\ge 0\) with \[\nu(A)=\int_A f\,d\mu \quad\text{for all }A\in\mathcal{E}\] We write \(f=\frac{d\nu}{d\mu}\) (the Radon-Nikodym derivative).

Exercise 10.5 proves the chain rule: if \(\nu\ll\mu\ll\omega\) with \(f=\frac{d\nu}{d\mu}\) and \(g=\frac{d\mu}{d\omega}\), then \(\frac{d\nu}{d\omega}=fg\).

Proof: For any \(A\in\mathcal{E}\), \(\nu(A)=\int f\cdot\mathbf{1}_A\,d\mu=\int fg\cdot\mathbf{1}_A\,d\omega\) by the change-of-measure formula (Prop 10.1.4).

Change of Variables (Exercise 11.4)

For a linear map \(\phi(x)=Ax+b\) with \(A\) invertible:

\[ \int_{\mathbb{R}^d} f(Ax+b)\,d\lambda(x) = |\det A|^{-1}\int_{\mathbb{R}^d} f(y)\,d\lambda(y) \]

Exercise 11.4(d): translate \(x\mapsto x-\xi\) and scale \(x\mapsto x/\sqrt{2\sigma^2}\) to derive \(\int e^{-(x-\xi)^2/(2\sigma^2)}\,d\lambda=\sqrt{2\pi\sigma^2}\).

11.1 Zero-One Laws (Exercise 13.3)

If \(P(X\in B)\in\{0,1\}\) for all \(B\in\mathcal{B}(\mathbb{R})\), then \(X\) is almost surely constant: \(\exists\, c\in\mathbb{R}\) with \(P(X=c)=1\).

Proof: \(F_X(t)=P(X\le t)\) takes only values \(0\) and \(1\). Since \(F_X(-\infty)=0\) and \(F_X(+\infty)=1\), there must be a jump at some \(c\). By right-continuity, \(F_X(c)=1\) and \(F_X(t)=0\) for \(t

11.2 Constructing Independent Random Variables (Exercise 13.5)

Given probability measures \(\mu_1,\ldots,\mu_n\) on \((\mathbb{R},\mathcal{B}(\mathbb{R}))\), construct a probability space with independent \(X_1,\ldots,X_n\) satisfying \(P_{X_j}=\mu_j\):

  • Take \(\Omega=\mathbb{R}^n\), \(\mathcal{F}=\mathcal{B}(\mathbb{R}^n)\), \(P=\mu_1\otimes\cdots\otimes\mu_n\), \(X_j=p_j\) (projection).
  • Independence: \(P(X_1\in A_1,\ldots,X_n\in A_n)=P(A_1\times\cdots\times A_n)=\prod\mu_j(A_j)=\prod P(X_j\in A_j)\).
